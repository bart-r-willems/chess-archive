# Public Chess Archive

This repository contains my complete collection of chess games, exported from ChessBase and stored as raw PGN files.

## Structure
- `pgn/` — All PGN files, organized by year and event.
- `viewer/` — A lightweight PGN viewer (pgn4web) for replaying games directly in the browser.
- `index.html` — A simple landing page linking to events and the viewer.

## How to View Games
You can open any `.pgn` file directly on GitHub, or use the interactive viewer:

- Open `viewer/viewer.html`
- Select a PGN file from the dropdown

## Why GitHub?
- High uptime
- Permanent URLs
- Version control
- Handles ChessBase PGNs without parsing issues
- Public and easy to share
